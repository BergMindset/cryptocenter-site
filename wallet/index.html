Кошелёк_

Вставь адрес — покажем активы по сетям прямо из блокчейна. Ничего не подключаем и не храним: только чтение, on-chain.

Данные: топ-сети через провайдера GoldRush (Ethereum · BNB · Polygon · Avalanche · Base · Arbitrum · Optimism и др., спам-фильтр), при сбое — резерв на открытых Blockscout. DeFi-блок распознаёт LP-токены пулов, стейкинг-деривативы (stETH), vault-шэры и aToken/cToken, которые лежат в кошельке. Концентрированные позиции Uniswap V3 / PancakeSwap V3 (это NFT) считаются он-чейн: суммы токенов из ликвидности и тиков, цены — из провайдера, со статусом «в диапазоне / вне диапазона». Застейканное без токена-расписки — следующий этап. Метки адресов и избранное — общие, в нашей базе. Не является финансовой рекомендацией.
